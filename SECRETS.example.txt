Do NOT commit real secrets.

Suggested GitHub Actions secrets (optional, if later needed):
- SB_CONNECTION_STRING
- SB_QUEUE_NAME

Current workflow does not inject secrets into artifacts automatically.
Reason: avoid baking production credentials into downloadable binaries/scripts.

Recommended deployment flow:
1. Download artifact
2. Edit run-windows.bat or run-macos.sh locally
3. Fill in SB_CONNECTION_STRING / SB_QUEUE_NAME / DEVICE_ID on the target machine
